Mouse events: click – when the mouse clicks on an element (touchscreen devices generate it on a tap). contextmenu – when the mouse right-clicks on an element. mouseover / mouseout – when the mouse cursor comes over / leaves an element. mousedown / mouseup – when the mouse button is pressed / released over an element. mousemove – when the mouse is moved. Form element events: submit – when the visitor submits a
. focus – when the visitor focuses on an element, e.g. on an . Keyboard events: keydown and keyup – when the visitor presses and then releases the button. Document events DOMContentLoaded – when the HTML is loaded and processed, DOM is fully built. CSS events: transitionend – when a CSS-animation finishes. There are many other events.
